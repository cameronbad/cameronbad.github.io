EduTest

A webapp I worked on as a collaborative project in a group of 4 during my final year of university. The brief given for this project was to "design, develop and test a multiple-choice testing platform", I acted as the group leader and was involved in the development of the student dashboard and the test functionality. The webapp is no longer hosted online due to having used university databases and servers, but the entire project was coded in GitHub and the repository is publically available.

Morsecode Translator

A C# project I created as part of my university course, it works entirely through the console window. The main functionality of this project is the ability to translate text into morse code and vice versa. It also had functionality for creating encoded morse code messages, alogn with a training function where you were shown a morse character and had to input the corresponding alphabetical character. The repository for this project is hosted publically on GitHub.
